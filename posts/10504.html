第九天笔记

前端项目学习

项目目录

  • xtx-pc
    • images文件夹：存放固定使用的图片素材，例如：logo、样式修饰图等等
    • uploads文件夹：存放非固定使用的图片素材，例如：商品图、宣传图、需要上传的图片
    • iconfont文件夹：字体图标素材
    • css 文件夹：存放CSS文件(link标签引入)
      • base.css：基础公共样式(最先引入)
      • common.css：各个网页相同模块的重复样式，例如：头部、底部
      • index.css：首页CSS样式
    • index.html：首页HTML文件

SEO（搜索引擎优化）

提升 SEO 的常见方法：

  1. 竞价排名
  2. 将网页制作成 html 后缀
  3. 标签语义化
  4. ……

网页头部 SEO 标签

  • title：网页标题标签
  • description：网页描述
  • keywords：网页关键词

Favicon 图标

Favicon图标：网页图标，出现在浏览器标题栏，增加网站辨识度。

图标：favition.ico 一般存放到网站的根目录里面

版心

wrapper 版心宽度：1240px

common.css 设置即可

导航

结构：通栏 > 版心 > 导航 ul

布局：flex-end

头部

结构：.header > logo + 导航（nav） + 搜索（search） + 购物车（cart）

注意

  • logo：使用 h1 > a，a放关键字并设置字号 0px 来提升搜索引擎优化
  • 导航：下边界线加给 a 而不是 hover，确保悬停时从下间距开始显示边界线
  • 搜索：浏览器解析 input 优先生效自己的默认宽度，宽度设置为 0 重置宽度使 flex: 1生效
  • 购物车：子绝父相中 left 默认向右撑开，right默认向左撑开
